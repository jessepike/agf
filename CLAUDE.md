# AGF — Agentic Governance Framework

Reference architecture and operating model for organizations governing agentic systems. Synthesizes NIST, OWASP, CSA, ISO, EU AI Act, OpenTelemetry, and academic research into a coherent playbook.

## Key Locations

- **Framework docs:** `docs/` (20+ MD files — governance framework, primitives, decision intelligence, observability)
- **Architecture diagrams:** `diagrams/` (21 diagrams)
- **Research archive:** `.private/research/`
- **Decision log:** `DECISIONS.md`
- **Published site:** agf.jessepike.dev

## Project Constraints

- This is a published reference architecture — treat content changes as editorial decisions, not code changes
- All factual claims must be substantiable with cited sources
- Content production happens via pike-acm (Campaign 03), not directly in this repo
- Intent document is the north star — read it before making framework changes
